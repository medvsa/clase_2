Hola JavaScript!

Además de los números enteros (int), están los decimales (float):

var a = 97.7

También están las cadenas de caracteres (string) que las podemos entender como letras, palabras o frases:

var a = "Radio Zero"

Otro modo de almacenar datos es el arreglo (array), que puede ser un conjunto de números, cadenadas de caracteres e incluyo otros arreglos:

var a = ["Radio Zero", 97.7]

JavaScript también la notación de Objeto (object), que nos permite identificar a cada contenido por el nombre al que se le asocia.

var a = {rol:"Radio", nombre:"Zero", dial:97.7}

En cada caso estoy creando y asignando datos a una variable de nombre a. Pero podría estar usando cualquier nombre con palabras (sin tildes ni acentos), evitando aquellas reservadas por JavaScript.
